# ReactAdminExample

ReactAdmin を用いた管理画面サンプル

### 環境構築

## 1. `git clone`

```bash
git@github.com:fjwr1024/react-admin-sample.git
```

## 2. `yarn install`

`yarn install`

```shell
# v1.22.15以上であることを確認
yarn -v

# パッケージをインストール
yarn install
```

## 3. `yarn start`

`yarn start`でローカルサーバーを立ち上げ。

```bash
yarn start
```
